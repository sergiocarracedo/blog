Con más de 20 años de experiencia—desde mi primer **Amstrad CPC464** hasta la construcción de modernas **arquitecturas impulsadas por IA**—resuelvo problemas combinando arquitecturas limpias con automatización práctica.

---

## 🤖 IA e Innovación

No solo converso con la IA; construyo sistemas que la integran en el ciclo de vida del desarrollo.

- **[storybook-to-skill-md](https://github.com/sergiocarracedo/storybook-to-skill-md):** Genera automáticamente archivos de habilidades para agentes de IA a partir de stories de componentes **React/Vue**, convirtiendo documentación de **Storybook** a formato **SKILL.md** listo para LLMs. Incluye herramienta CLI y **GitHub Action**.
- **[Work Diary](https://github.com/sergiocarracedo/work-diary):** Herramienta CLI de código abierto para automatizar resúmenes diarios de trabajo usando **Gemini/OpenAI** y la actividad de GitHub.
- **[Sireno AI](https://sergiocarracedo.es/blog/sireno-ai-assistant/):** Extensión de navegador que lleva corrección de texto y traducción con IA a cualquier formulario web.
- **Visión por Computador:** PoC temprana de fondos virtuales por webcam usando **TensorFlow.js** (`tfjs-bodypix`).
- **IA para CAD:** Uso de LLMs para asistir en el diseño de objetos impresos en 3D.

## 🏗️ Design Systems & UX

Experto en la creación de bibliotecas de UI escalables y visualizaciones de datos complejas.

- **Factorial `f0`:** Colaborador principal y tech lead (hasta marzo de 2026) del **Factorial Zero Design System**, construyendo componentes robustos en React/TypeScript.
- **[SirenoGrid](https://github.com/sergiocarracedo/sireno-grid):** Creador de un framework CSS Grid ligero con fallbacks en flexbox.
- **Visualización de Datos:** Implementación avanzada de **ECharts** para productos de machine learning y análisis.
