Con más de 20 años de experiencia —desde mi primer **Amstrad CPC464** hasta la construcción de **arquitecturas modernas con IA**— resuelvo problemas combinando arquitecturas limpias con automatización práctica.

## 🤖 IA e Innovación

No me limito a conversar con IA; construyo sistemas que la integran en el ciclo de desarrollo.

- **[storybook-to-skill-md](https://github.com/sergiocarracedo/storybook-to-skill-md):** Genera automáticamente archivos de skill para agentes IA a partir de stories de **React/Vue**, convirtiendo documentación de **Storybook** en formato **SKILL.md listo para LLMs**. Incluye CLI y **GitHub Action**.
- **[Work Diary](https://github.com/sergiocarracedo/work-diary):** Herramienta CLI open-source para automatizar resúmenes diarios de trabajo usando **Gemini/OpenAI** y actividad de GitHub.
- **[Sireno AI](https://sergiocarracedo.es/blog/sireno-ai-assistant/):** Una extensión de navegador que lleva corrección de texto y traducción potenciadas por LLMs a cualquier formulario web.
- **Computer Vision:** PoC temprano de fondos virtuales por webcam usando **TensorFlow.js** (`tfjs-bodypix`).
- **IA para CAD:** Uso de LLMs para asistir en el diseño de objetos físicos impresos en 3D.

## 🏗️ Design Systems y UX

Experto en la creación de bibliotecas de UI escalables y visualizaciones de datos complejas.

- **[SirenoGrid](https://github.com/sergiocarracedo/sireno-grid):** Creador de un framework CSS Grid ligero con fallbacks de flexbox.
- **Data Viz:** Implementación avanzada de **ECharts** para productos de machine learning y analítica.
