Nowadays, most users have several browser tabs open simultaneously while browsing. Sometimes our web application needs to get the user's attention to require an action or simply let them know they have received a notification.

A good way to do this is to alternately change the page title with an alert, just like Facebook does when we receive a chat notification; the page title changes 3 or 4 times between "Facebook...." and "So-and-so has sent you a message."

To facilitate this seemingly simple task, we have created a very easy-to-use jQuery plugin.

We just need to pass the alert message that will alternate with the original page title as a parameter.

![](/images/chrome-blank-new-tab.gif)

We can pass the number of repetitions and the time (in milliseconds) between changes as options.

Here is the source code for the plugin:

jQuery.extend({
  titleBlink : function(title,options) {
    var defaults = {
      repeat : 5,
      delay : 800
    };
    var options = $.extend(defaults, options);
    var repeatCount = 0;
    var currentTitle=$(document).attr("title");

    var blinkInterval = setInterval(function() {
      if($(document).attr("title")==currentTitle) {
        $(document).attr("title",title);
      } else {
        $(document).attr("title",currentTitle);
        repeatCount++;
        if (repeatCount==options.repeat) {
          clearInterval(blinkInterval);
        }
      }
    }, options.delay);
  }
})
